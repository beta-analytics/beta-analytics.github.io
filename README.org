  Changes made: (just followed the refrence url, everything supereasy)

      | domains                |
      |------------------------|
      | www.beta-analytics.com |
      | beta-analytics.com     |

    - create a dns entry to use the gitlab static page server: 
      #+BEGIN_EXAMPLE
      mydomain.com A 52.167.214.135
      #+END_EXAMPLE
